# README（Backend）

### 一、项目依赖

# 核心运行依赖
fastapi>=0.104.1          # 高性能Web API框架，用于接口开发与请求处理
uvicorn>=0.24.0           # ASGI服务器，负责运行FastAPI应用
sqlalchemy>=2.0.23        # ORM框架，简化数据库操作，实现数据持久化
pydantic>=2.4.2           # 数据校验与格式化，规范接口入参/出参
python-jose>=3.3.0        # JWT令牌生成与解析，用于用户身份鉴权
passlib>=1.7.4            # 密码加密库，采用bcrypt算法存储密码

# 数据库（无需额外安装）
sqlite3                   # 内置轻量数据库，自动生成数据库文件，无需手动配置


### 二、项目架构

backend/
├── __init__.py            # Python包标识文件，标记backend为可导入包
├── database.py            # 数据库核心配置，创建数据库引擎、会话，提供数据库连接依赖
├── models.py              # ORM数据模型，定义用户表（users）、任务表（tasks）结构及关联关系
├── schemas.py             # 接口数据模型，定义接口入参、出参格式，规范数据交互
├── security.py            # 安全认证相关，包含密码加密、JWT令牌生成与验证逻辑
├── crud.py                # 数据操作层，实现用户、任务的增删改查核心逻辑
└── README.md              # 后端项目说明文档（本文档）

### 三、运行方法

## 1、环境准备
确保已安装 Python 3.9+，打开命令行，进入项目根目录（app 与 backend 文件夹同级目录），执行以下命令安装所有依赖：
bash
运行
pip install -r requirements.txt
## 2、启动后端服务
在项目根目录执行以下命令，启动开发模式（自动热重载，修改代码无需重启服务）：
bash
运行
uvicorn app.main:app --reload
## 3、访问与调试
后端服务地址：http://127.0.0.1:8000
API 调试文档（自动生成，可直接测试接口）：http://127.0.0.1:8000/docs
数据库文件：启动服务后自动生成 task_manager.db（SQLite），无需手动建库建表
数据持久化：关闭服务后，数据库数据不会丢失，重启服务可继续使用


### 四、补充说明
本文件夹为后端核心逻辑层，负责数据处理、身份鉴权、数据库操作，不包含前端页面相关代码；
接口鉴权：所有任务相关接口需携带 JWT 令牌，仅允许操作当前登录用户的任务数据，保证数据隔离；
依赖说明：requirements.txt 需放在项目根目录，与 app、backend 文件夹同级，用于统一管理依赖包；
调试提示：若接口调用失败，可先通过 API 调试文档（/docs）测试接口可用性，排查参数或令牌问题。
